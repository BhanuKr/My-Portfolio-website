    Projects

    Rental Bike Count Prediction

    Developed a predictive model using XGBoost to forecast bike rentals based on various factors. Optimized model performance through feature engineering and hyperparameter tuning with Optuna.

    Conducted A/B testing to analyze rental patterns, using statistical tests like t-tests, ANOVA, and Chi-squared tests to identify key influences on bike demand.

    Implemented data preprocessing and visualization to clean and transform data, ensuring better model accuracy. This project demonstrates the use of machine learning and statistical analysis to improve demand forecasting and decision-making in urban mobility.

    Arched Eyebrows Detection using ResNet18

    Developed a deep learning model using ResNet18 to classify facial images based on the 'Arched Eyebrows' attribute. Trained on a subset of the CelebA dataset, the model utilizes PyTorch and is evaluated with accuracy, precision, recall, F1 score, and the ROC curve.

    Implemented data preprocessing and dataset splitting into training, validation, and test sets. Fine-tuned hyperparameters to optimize model performance.

    This project demonstrates the application of computer vision and AI-based facial attribute recognition to enhance automated image analysis.
